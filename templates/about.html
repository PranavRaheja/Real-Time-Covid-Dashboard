About COVID Statistics Dashboard

Project Overview

This dashboard provides real-time COVID-19 statistics from around the world. The project was developed to help visualize and understand the impact of the pandemic through accessible and interactive data representation.

Features

  • Dark Mode: Toggle between light and dark themes
  • Accessibility: Screen reader support, keyboard navigation
  • Search: Find statistics for any country
  • Regional Analysis: Compare statistics across continents
  • Data Export: Download data in CSV or JSON format
  • Responsive Design: Works on mobile and desktop

Data Source

All COVID-19 statistics are sourced from Worldometers, which aggregates data from official government reports and international organizations.

Accessibility Features

This dashboard is designed with accessibility in mind:

  • Screen reader announcements for key actions
  • Keyboard shortcuts (Alt+T: theme, Alt+H: home, Alt+S: search)
  • Skip links for keyboard navigation
  • High contrast mode support
  • Reduced motion support
  • Proper ARIA labels and roles

Keyboard Shortcuts

  • Alt + T: Toggle dark/light mode
  • Alt + H: Navigate to homepage
  • Alt + S: Focus search input
  • Tab: Navigate through interactive elements
  • Enter/Space: Activate buttons and links
